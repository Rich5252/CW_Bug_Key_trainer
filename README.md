This an on-going development of a CW Trainer app focussed on improving sending with a mechanical bug key.

It shows timing and decode with analysis of timing accuracy.

All aimed at improving your bug key sending skill and consistancy.

It is also useful for tweaking the key settings for better timing.

Let me know if this app sounds useful and if you would like to see specific features.
The mechanical bug is new to me so still learning.

Currently this app is driven from a USB COM port from my bug-key sidetone module.
I will add audio input later so let me know if this would be useful to you.
